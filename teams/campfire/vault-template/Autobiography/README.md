# Autobiography

Your life story lives here.

- **Long-form chapters** go in this folder (one note per chapter/era).
- **Individual memories** go in `Stories/` (one event per note) and link up to the
  relevant chapter and to `[[_MOC/Autobiography MOC]]`.

Start by talking to the **Biographer** (`/campfire` → "I want to tell a story
about …"). It will interview you, draft the note, read it back, and file it here
on your OK. Duplicate `Stories/_TEMPLATE.md` for a manual start.

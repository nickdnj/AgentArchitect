# _changelog — edit audit trail

Because this vault may have **no cloud backup** while you travel, every time an
agent changes an *existing* note it records the change here, so any edit can be
undone by hand. Creating a brand-new note does **not** get logged.

## Format
One file per day: `YYYY-MM-DD.md`. Append newest at the bottom.

```markdown
## HH:MM — Autobiography/Stories/the-maine-trip.md  (memory-keeper)
- **Change:** corrected the year of the Maine trip
- **Old:** "the summer of 1989"
- **New:** "the summer of 1987"
- **Reason:** Nick corrected the date
```

## Who writes here
- **Biographer** — when it expands an existing story.
- **Memory Keeper** — when it corrects a fact or reconciles a contradiction.
- **Vault Librarian** — when it changes a note's body (link/tag/MOC edits).

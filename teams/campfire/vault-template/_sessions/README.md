# _sessions — continuity log

This team has no wiki and no cloud, so session continuity lives right here in the
vault. After each working session the Campfire orchestrator appends a short entry
so the next session knows where you left off.

## Format
One file per day: `YYYY-MM-DD.md`, newest at the bottom.

```markdown
## 20:10 — campfire
- **Asked:** capture the story about the Maine road trip
- **Specialists:** biographer
- **Output:** created Autobiography/Stories/the-maine-trip.md (status: draft)
- **Open follow-ups:** confirm the mechanic's name; expand the diner scene
```

You can read this anytime to recall open threads — or ask the team "what were we
working on?" and it will check here.

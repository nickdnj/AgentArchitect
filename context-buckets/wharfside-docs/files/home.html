Welcome to the Transcribed Documents Website

This website displays transcriptions of scanned documents that have been processed using an advanced transcription model (GPT-4o).

How It Works: The model transcribes all visible printed text from each scanned page. Handwritten annotations—such as dates, checkmarks, or initials—are enclosed in curly brackets. For example, a handwritten "J.D." appears as {J.D.}.

If the model is uncertain about any part of the text, that portion is highlighted in red. This red highlight serves as a cue for you to refer to the original scanned image for verification.

Additionally, tables and bold text are formatted using simple HTML tags to retain their structure and emphasis.

Use the sidebar on the left to navigate among the documents. Click "Home" at any time to view this information again.

## Part 1 — The Museum in Context

### Where it is

The InfoAge Computer Museum is one of several museums located on the **Camp Evans / Marconi Belmar Site** at Wall Township, NJ. The campus today is operated by **InfoAge Science & History Center**.

### Site history (from the Radio Tech Museum handbook, Section 4)

A surprisingly layered location:

| Era | Operator | Notes |
|---|---|---|
| **1913–1914** | **Marconi Company of America** | Built the original Belmar receiving station to communicate with Carnarvon, Wales. Five buildings + six 400-foot masts. Paired with a transmitter site in New Brunswick, NJ. |
| **WWI (1917+)** | **US Navy** | Took over Belmar (and New Brunswick) for the duration. |
| **1919–1920** | (Brief Marconi return that didn't happen) | US wanted to keep the new **Alexanderson Alternator** out of British hands. |
| **1920–1926** | **Radio Corporation of America (RCA)** | Owen Young (GE) chartered RCA to consolidate radio. Marconi was forced to sell its US holdings to RCA. RCA operated Belmar until 1926, then consolidated everything at "Radio Point" on Long Island and abandoned Belmar. |
| **1926–1936** | (Various non-radio uses) | Briefly used by the **Monmouth Pleasure Seekers** — a local KKK chapter. |
| **1936–1941** | **The Kings College** | Small Christian college converted the buildings into a campus. Outgrew the site by 1941. |
| **1941–1996** | **US Army Signal Corps — Camp Evans** | Bought property Sept 11, 1941; war declared Dec 11. **Did secret radar research.** Famous "radar was a classified word — destroy all signs that say 'radar development lab'" incident. Ran from WWII through Vietnam through Cold War. |
| **1996** | **BRAC closure** | Base Realignment and Closing Commission shut Camp Evans. Property transferred to Wall Township. |
| **1996–today** | **InfoAge** | Multi-museum science and history center. |

The Computer Museum is operated by the **Vintage Computer Federation (VCF)**, a 501(c)(3) charity. The Radio Tech Museum next door is operated by the **New Jersey Antique Radio Club (NJARC)**. They share a campus and some facilities (HVAC, hallways, storage).

### How VCF talks about itself

- **VCF** = Vintage Computer Federation, vcfed.org
- Holds **VCF East**, an annual computer festival at InfoAge (Apr 17–19, 2026 was the most recent)

**UNIVAC 1219-B (military computer, 18-bit)**
- ✅ Detailed narrative: Terrier Missile Fire Control System, Mk 19 DDR, 1541 tape drive, 32×104 parameters per second saved for analysis, classified tactical program
- 1540/1541 reads at 120 in/sec; tension arms compensate for sudden direction changes; .75" inter-record gap
- **Has a video:** "The Three T's Missile" — TV with remote located on paper tape reader
- ✅ Has the original **Sperry Rand UNIVAC 1219 General Description brochure** (PX 3678, ~1968):
  - Medium-scale general-purpose military, faster version of 1218, MIL-E-16400 spec
  - 18-bit one's complement, 500 kwords/s, 102-instruction repertoire
  - Cabinet: 71.75" × 26.25" × 30.5", 1000+ lbs, 2000W @ 115V/3-phase/400-cps
  - Memory: 2μs main core (8K–64K 18-bit words), 500ns control core
  - Software: TRIM I/II/III, Fortran IV, CS-1, Floating Point Package
  - **Used in NTDS** (Naval Tactical Data System)
- Light in cabinet inconsistently working (recurring meeting note)

**Wang 4000** (1967, An Wang)
- ✅ Polished sign + ✅ donor provenance (Frank DeLorenzo) + ✅ 121 photos including **systematic chassis-card catalog** (W4001_A_01a..21b, W4001_B_01a..24b, both sides) + ✅ handwritten **operator cheat sheets** (8 cards + 9 candid photos taken 2022-07-31)
- ✅ Original instruction manual + Wang 300/320E card layout PDFs (2 copies each)
- An Wang patented core memory 1955, sold to IBM for $500K in 1956
- 4000 patent: **US 3,470,542 A** to Frank Trantanella (modular design)
- 1KB memory = 8× 32×32 core modules
- ROM board = matrix of diodes at crosspoints (~13× 15-bit quantities)
- Logic boards are **discrete transistors and diodes hand-soldered on perfboard** — confirmed via chassis-card photos (e.g., W4001_A_01a card #23 marked 2218-3T)
- LOCI-2 calculator predecessor shown in sign
- **Above the Wang 4000:** items including a "Mother of All Demos" video (per 2024-10-10)

**RCA COSMAC 1802** (1975, Joseph Weisbecker)
- ✅ Polished sign (5K transistors, first CMOS low-power CPU, ~1600 inst/sec, still in production)
- Connected items: RCA Studio II console (1977), Galileo spacecraft (orbited Jupiter 8 yrs), Chrysler Spark Control 1970s, RCA COSMAC VIP (CHIP-8 — first game "engine"), 1976 Popular Electronics "COSMAC ELF" build
- **The PDP-8 lock-box password is the COSMAC model number: 1802** (operational trivia)

## Part 7.5 — Extended findings (v1.1, after deep PDF pass + video transcripts)

### Wang 4000 — full architectural picture

The 1967 **Wang 4000 Reference Manual** (Wang Laboratories Inc., 836 North Street, Tewksbury, MA 01876, September 1967) yields a complete operational picture:

- **5-chassis modular system** (rack-mounted, each 7"×19"×18", except keyboard 12"×6"×14"):
  - **Model 4001 Arithmetic Unit** — internally identical to Wang 320 desk calculator. 4 registers (LA/RA/W/L). 14 four-bit BCD digits per register. **Built-in functions: +, −, ×, ÷, x², √x, logₑ x, eˣ** — the Wang 4000 had logarithms and exponentials in 1967. This is a real talking-point.
  - **Model 4002 Keyboard** — full layout shown in Figure 7.3, has every instruction key (CCS/ICS/OCS/JMP/TRA/LSA/TTC/TSW/EOT/LDW/CHANGE SIGN/CLEAR W/CLEAR ALL/STROBE/ACK/STEP/CTU/STORE/READ/RECALL/WRITE/CLEAR ADDER/RECALL ADDER) plus number pad and function keys
  - **Model 4006 Teletype Control** — converts 8-bit busline → 11-unit serial for Teletype Model 33 ASR
  - **Model 4007 Memory** — magnetic core, 1024 8-bit characters base (expandable to 4096); 128 sixteen-digit registers possible
  - **Model 4008 Memory Control** — busline ↔ memory routing
- **Busline = 24-wire cable**, 8-bit binary codes. Voltage: 0V = binary 0, **−10V = binary 1** (negative logic). Octal weighting on lines 1-8: 1, 2, 4, 10, 20, 40, 100, 200. Strobe (line 13) + Acknowledge (line 14) + Prime (line 9, clear) + Continue (line 11) + Sign (line 12).
- **Channel addresses:** keyboard=0, arithmetic=1, memory=7, teletype=6
- **Memory in octal:** 1024₁₀ characters = 2000₈ locations; storage register 0 (S0) at 1770₈-1777₈ (registers grow downward from top of memory)
- **Power:** Single 115V 60-cycle single-phase per chassis. *"Power required by the system circuitry is less than that required by a few household lamps. There are virtually no parts which will wear out. The teletype machine should, however, be turned off when not in use."* — original guidance.

### Wang 4000 — operating procedure (docent demo recipe)

**To start a stored program at address [XX]:**
```
CLEAR ALL
CCS 7              (set memory as control)
CSS [XX]           (location of first instruction, 1-4 digits)
CTU                (start)
```

**To load a new program from keyboard:**
```
CLEAR ALL
ICS 0              (select keyboard as input)
OCS 7              (select memory as output)
OSS [XX]           (location of first instruction)
TRA                (transfer; now I lamp on KB and OUTPUT lamp on memory control)
[key in program]
EOT                (end)
```

**To dump memory to paper tape:**
```
ICS 7
ISS [start_addr]
OCS 6
TRA
```

**Display feedback:** MEMORY ADDRESS lamp shows current address. MEMORY BUFFER shows next instruction. STEP-AUTO toggle for single-step debug. **CLEAR ALL must NOT be used while AUTO is running** — will lose program info.

### Wang 4000 cheat sheet finding (the heat-exchanger sheet)

Among the photographed cheat sheets, one (`IMG_20220731_134139492.jpg`) is **NOT a Wang 4000 reference**. It's a hand-written engineering reference card for **shell-side pressure drop calculations in heat exchangers** — Reynolds numbers, friction factors, finned-tube and plain-tube formulas, with variables P (tube pitch), D (O.D.), gₐ (gravitational constant 4.18×10⁸ ft/hr²ft²), ρ (density lb/cu ft), µ (viscosity at tube wall in lb/ft·hr).

**Implication:** This is not Wang's documentation — it's the **original chemical engineering work product of Frank DeLorenzo's father's chemical company in NJ** (the donor; per the Wang 4000 provenance doc, "the computer belonged to a chemical company here in NJ. They bought it around 1967-1968"). The Wang 4000 was used to evaluate heat-exchanger designs. **This is the strongest possible authentication of the Wang 4000's actual industrial use.** Worth elevating to a primary visitor talking point.

### HP 1000 — full historical and engineering context

From the **March 1977 HP Journal** issue (~30pp, ~half visually read) and the **HP 1000 E-Series Tech Data sheet** (5pp, fully read):

- **Family lineage:** HP's first "instrumentation computer" was the **Model 2116A in 1966**. By March 1977, "the majority of over 16,000 2116's, 2100's, and 21MX's installed have been for scientific and engineering computation, data acquisition and control, factory automation, process control, product and production testing, and computer-aided design."
- **The HP 1000 line is NJ-relevant in spirit** — built in Cupertino CA but the first-customer "instrumentation computer" use cases (data acquisition, real-time process control, ATE) were exactly the kind of work Camp Evans / Fort Monmouth had been doing for the Signal Corps decades earlier.
- **E-Series specs (Models 2109E and 2113E):**
  - 2109E: 64KB std memory, expandable to 640KB; 9 I/O channels in 8¾" mainframe
  - 2113E: 128KB std, expandable to 1280KB; 14 I/O channels in 12¼" mainframe
  - 350 ns memory cycle time (high-performance memory option); 16-bit data path; 24-bit control path
  - 128 standard instructions; 211 control-processor instructions; up to 5 may be combined in 1 instruction
  - 2.28M byte/sec DMA via DCPC (Dual Channel Port Controller)
  - **User-microprogrammable** via WCS (Writable Control Store) — up to 3K user microinstructions
  - Memory parity check + fault control; Dynamic Mapping System (DMS) for 1.8MB; 50 distinct interrupt levels
  - Power: 88-132V/176-264V switchable, 47.5-66Hz, 625W max. Heat: 580 kcal/hour. Air flow: 200-280 CFM.
  - Weight: 45 lb (2109E) / 65 lb (2113E)
- **Software family:** **RTE-II / RTE-III** (Real-Time Executive) operating systems, FORTRAN IV, HP BASIC, Assembler, **IMAGE/1000** database, **DS/1000** distributed systems, multi-user real-time BASIC
- **Key 1977 personnel** (from HP Journal article bylines and bios):
  - **Lee Johnson** — joined HP 1966; wrote BCS for 2116A; managed RTE OS development; managed IMAGE database; system manager for HP 1000; engineering manager 21MX/RTE Lab; UNM math '63, MBA Santa Clara '70.
  - **David L. Snow** — RTE-II/III project manager; UT Austin BSEE; production engineer for RTE-II/TODS-C; later MSEE Stanford; member Air Force Reserve.
  - **Kathleen F. Hahn** ("Kathy") — On-Line Generator and SWTCH; Iowa State BSCS '73; husband also at HP; lived Sunnyvale CA.
  - **Harris Dean Drake** — Microprogramming package (microassembler, MDE microdebug editor, WCS load utility, PROM tape generator); UC Davis BSEE '69; US Army stretch; CPU/memory diagnostics for 21MX; backpacker, handball player.
  - **John Moll** — IC technology references; faculty Ohio State + Stanford; consultant to HP early semiconductor; Tech Director IC Lab.
- **Pricing in 1977 USD:** Model 30 = $36,500 / Model 31 = $31,500 / Model 80 = $62,000 / Model 81 = $62,000.
- **Manufactured at:** Data Systems Division, 11000 Wolfe Road, Cupertino, California 95014.

Source material and original drafts for the InfoAge Computer Museum docent manual.

## Layout

```
infoage-docent/
├── bucket.json              Metadata + key-asset index
├── README.md                This file
├── files/                   Read-only mirror of the Drive folder
│   └── Docent Shared Resources/
│       ├── Artifacts/       Per-machine subfolders (24 machines)
│       ├── Signs/           Existing exhibit signs (PDFs)
│       ├── Manuals/         Radio Technology Museum handbook (template)
│       ├── Reference/       Reference websites + Univac-1219 doc

│       ├── Demos/           Amiga tested-disks list
│       ├── Repair Information/
│       ├── Opening/         Opening event photos
│       ├── Meetings/
│       ├── Videos/          ~6 GB
│       └── (loose docs)     FAQ, checklists, plug map, schedule

    ├── operational/         Open/close, safety, dress, FAQ

```
